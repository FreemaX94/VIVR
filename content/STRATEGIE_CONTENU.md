## 1. CONTENU PRODUIT

### 1.1 Descriptions de produits actuelles

**Problemes identifies :**
- Descriptions trop courtes et generiques ("Lampe design scandinave en bois naturel")
- Manque de details techniques (dimensions, materiaux, entretien)
- Absence de storytelling et d'emotion
- Pas de mise en situation d'usage

### 1.2 Modele de description produit ameliore

```markdown
## Structure recommandee pour chaque produit

### Accroche emotionnelle (1 phrase)
[Phrase qui evoque un benefice lifestyle]

### Description principale (2-3 phrases)
[Description detaillee avec storytelling]

### Caracteristiques cles (puces)
- Materiau : [detail]
- Dimensions : [L x l x H cm]
- Poids : [kg]
- Couleur : [nom evocateur]
- Style : [scandinave, boheme, industriel...]

### Conseils d'utilisation
[Comment integrer le produit dans son interieur]

### Entretien
[Instructions de nettoyage/maintenance]
```
